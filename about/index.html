this site

This site is built using a JAMstack:

  • Javascript -- NodeJS + GatsbyJS pre-rendering with a progressive SPA
  • APIs -- a self-hosted Netlify webapp leverages a Git backend on Github to provide a CMS
  • Markup -- there's markup involved! Lots of it! The build process consumes markdown from the CMS and outputs HTML.
